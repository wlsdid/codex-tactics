# Study Note - Portfolio Capture Folder

A portfolio project benefits from predictable evidence locations.

## Why prepare the folder early?

Screenshots and GIFs are not code, but they are part of the portfolio review experience. If the folder is prepared in advance, the capture workflow is simple:

1. Run Unity validation.
2. Capture screenshots/GIFs.
3. Save them under `Docs/Captures/`.
4. Link them from README.

## Good capture filenames

Good filenames describe the visible proof:

- `01_battle_start.png`
- `02_fire_skill_burn.png`
- `03_guard_status.png`
- `04_result_summary_rank.png`
- `05_retry_reset.png`
- `codex_tactics_battle_loop.gif`

This makes it easy to understand what each file proves without opening it.

## Portfolio lesson

Code shows implementation. Captures show player-facing results. A strong game portfolio needs both.
