# Study Note — Unity UI Slider HP Bars

## Concept

A Unity `Slider` can be reused as a read-only HP bar by setting:

```csharp
slider.minValue = 0f;
slider.maxValue = maxHp;
slider.value = currentHp;
slider.interactable = false;
```

## Applied in this project

`BattleManager.UpdateUI()` now updates both HP text and HP sliders. This keeps visual UI and numeric UI synchronized from one battle-state update path.

## Builder pattern

The scene is regenerated from `BattleSceneAutoBuilder`, not manually edited as a scene binary. The builder creates:

- `Player HP Slider` under player HP text
- `Enemy HP Slider` under enemy HP text
- fill images with different colors for readability
- serialized references into `BattleManager`

## Lesson

When UI shows the same game state in multiple ways, update it from one method to reduce mismatch bugs.
