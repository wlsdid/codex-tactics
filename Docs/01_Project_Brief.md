# 프로젝트 브리프

## 목적

대학교 과진입을 위한 게임 포트폴리오 제작.

## 장르

- 2D 픽셀 턴제 RPG
- 스테이지 클리어형 구조

## 왜 턴제 RPG인가?

- C#과 객체지향 구조를 배우기 좋다.
- 캐릭터, 스킬, 적, 상태이상, 스테이지를 데이터로 설계할 수 있다.
- 전투 UI와 밸런싱 과정을 포트폴리오에서 설명하기 좋다.
- 실시간 액션보다 개발 범위 관리가 쉽다.

## MVP 목표

첫 목표는 거대한 RPG가 아니라 **작은 전투 하나의 완성**이다.

- 아군 1명
- 적 1명
- 기본 공격
- HP 감소
- 턴 전환
- 적 자동 공격
- 승리/패배 판정

## 확장 목표

- 아군 3명
- 적 6종
- 보스 1~2종
- 스테이지 5개
- AP 시스템
- Break 시스템
- 상태이상 4개
- 포트폴리오 README / Notion 개발일지 / 플레이 GIF
