# 포트폴리오 제출용 페이지 초안

## 프로젝트 소개

Codex Tactics는 대학교 과진입 포트폴리오를 위해 제작하는 픽셀 그래픽 턴제 RPG입니다. 작은 전투 프로토타입에서 시작해 AP, Break, 스테이지, 보스 패턴을 단계적으로 확장하는 것을 목표로 합니다.

## 보여주고 싶은 역량

- C# 객체지향 기초
- Unity 2D 전투 시스템 구현
- 상태 머신 기반 전투 흐름 설계
- 데이터 기반 캐릭터/스킬 구조
- UI/UX 구성
- 개발 과정 기록과 문제 해결 능력

## 핵심 시스템

1. 턴제 전투
2. AP 기반 스킬 사용
3. Break 시스템
4. 보스 패턴
5. 스테이지 클리어 구조

## 개발 과정에서 기록할 문제 해결 사례

- 전투 순서를 어떻게 관리했는가
- 캐릭터 데이터를 어떻게 구조화했는가
- UI와 게임 로직을 어떻게 연결했는가
- 밸런싱 수치를 어떻게 조정했는가
