# Codex Tactics

> 대학교 과진입 포트폴리오를 목표로 제작하는 **스테이지 클리어형 픽셀 턴제 RPG**입니다.

## 프로젝트 목표

C#과 Unity를 공부하면서, 전투 시스템·UI·데이터 설계·밸런싱 과정을 기록하고 완성 가능한 인디게임 퀄리티의 포트폴리오를 제작합니다.

## 한 줄 소개

스테이지를 클리어하며 동료와 스킬을 성장시키는 픽셀 그래픽 턴제 RPG.

## 핵심 시스템 예정

- 턴제 전투 상태 머신
- AP 기반 스킬 사용
- 약점 공격 누적을 통한 Break 시스템
- 캐릭터 역할 조합: 탱커 / 딜러 / 서포터
- 스테이지 클리어형 진행
- 데이터 기반 캐릭터·스킬 관리

## 기술 스택
